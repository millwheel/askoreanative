**Ask real Koreans. Get real answers.**

AsKoreaNative is a Q&A platform for foreigners living in or visiting Korea.  
Users can ask practical questions about Korean culture, daily life, travel, food, and safety,  
and receive answers directly from real Koreans with local experience.

Built with [Next.js](https://nextjs.org), the service focuses on simplicity, clarity, and trustworthy answers.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
